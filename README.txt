Adds an exhibition post type to WordPress that synchronizes with DigitaltMuseum.

== Description ==

Exhibition adds a custom post type to WordPress that fetches information through the DigitaltMuseum API.

== Installation ==

1. Upload the `exhibition` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
